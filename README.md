## Description

Backend of blog website using Nest.js and MongoDB.

## Live

[Live Swagger Link.](https://nest-blog-c34p.onrender.com/docs)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
